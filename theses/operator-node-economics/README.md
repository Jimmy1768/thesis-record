# Operator Nodes

This directory contains Thesis 1 migrated from the former
`operator-node-economics` repository.

The initial ThesisRecord migration preserves research notes and operator records
as historical artifacts. Some files may still reference the former
`living_dissertation_app` name or Living Dissertation phase labels because they
record prior work. Current platform code lives under `thesis_record_app/`.

`paper/draft.md` is intentionally held back from the initial migration.

## Current Phase

Operator Nodes is the Phase 1 thesis. The near-term goal is to publish a v0
thesis record, freeze the checkpoint timeline, and run controlled evidence
collection/assimilation jobs toward 3, 5, and 10 year milestones.

This phase is about making Thesis 1 durable and measurable. Public
multi-workspace productization is Phase 2.
