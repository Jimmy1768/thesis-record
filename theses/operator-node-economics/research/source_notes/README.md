# Source Notes

Create one note per source.

Suggested filename:

```text
author-year-short-title.md
```

Each note should include:

- bibliographic reference;
- source type;
- core claims;
- evidence used;
- relevance to the hypothesis;
- objections or limitations;
- useful quotations with page numbers where available;
- related claims in `research/claims_index.md`.

Do not turn source notes into conclusions.
