# Operator Records

This folder stores repo-local coordination records for the research workflow.

- `handoffs/` - scoped work packets for research or writing threads.
- `returns/` - completed work returns.
- `acceptances/` - reviewed decisions on returns.
- `execution_records/` - durable closeout records.

The Company Operator owns record discipline. Research and writing threads should
not treat unaccepted returns as source truth.
