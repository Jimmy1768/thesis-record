# Evidence Snapshots

Status: future checkpoint records. Not active thesis verdicts.

Use this folder for frozen evidence-state records tied to scheduled forecast
versions such as V1, V2, V3, and V4.

Snapshots should capture:

- code/version reference;
- source manifests used;
- reviewed observations or exports used;
- claim-review state;
- prediction status at the checkpoint;
- unresolved gaps;
- disconfirming evidence;
- publication/checkpoint decision.

Do not place raw data, private data, local database dumps, or generated analysis
outputs here unless a separate export policy explicitly authorizes a tracked
artifact.
