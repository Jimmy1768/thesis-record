# Rejected Evidence

Status: audit trail for unusable or out-of-scope intake.

Use this folder when an item was inspected or triaged and should not support the
thesis.

Common rejection reasons:

- no primary source found;
- exact claim contradicted by source;
- source is unreliable;
- duplicate of a better source;
- outside thesis scope;
- arXiv-only or unreviewed source not accepted for this use;
- marketing claim without evidence;
- chart/image cannot be traced.

Rejected records should explain what was checked and why the item is not usable.
They should not become claim support or paper evidence.
