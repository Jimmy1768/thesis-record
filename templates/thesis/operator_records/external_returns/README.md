# External Returns

This directory stores raw external research returns for this thesis.

Raw returns are operator/process artifacts. They are not evidence, source truth,
accepted methodology, accepted scoring, accepted claims, or publication prose.

Use this directory for unedited outside returns from tools, consultants, Deep
Research, external query tests, or other non-authoritative research processes.

Each raw return should receive a separate intake note under:

```text
theses/<slug>/research/intake/
```

Recommended raw return filename:

```text
YYYY-MM-DD-<topic>-raw-return.md
```

Do not delete weak or rejected raw material merely because it was not accepted.
Rejected material remains part of the audit trail.

See:

```text
docs/source-verification/raw_return_buffer_policy.md
```
