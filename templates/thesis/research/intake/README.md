# Intake Notes

This directory stores interpreted intake notes for raw external returns.

Intake notes classify raw material before anything can be promoted into source
notes, methodology records, open questions, rejected material, claims, or prose.

Intake notes are not evidence. They are not source truth. They do not make an
outside return authoritative.

Recommended intake filename:

```text
YYYY-MM-DD-<topic>-intake.md
```

Use `raw_return_intake_template.md` for new intake notes.

See:

```text
docs/source-verification/raw_return_buffer_policy.md
```
